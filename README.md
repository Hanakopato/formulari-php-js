# Formulari de contacte interactiu (PHP + JS)

## Descripció
Formulari dividit en pestanyes (Personal, Mensaje, Revisar) amb:
- Validació en client (JavaScript pur) i en servidor (PHP).
- Enviament asíncron sense recarregar la pàgina (AJAX amb fetch).
- Emmagatzematge flexible: fitxer JSON o base de dades MySQL.
- Feedback visual immediat del resultat de l’enviament.

## Tecnologies
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: PHP 7.4+
- **Base de dades**: MySQL (opcional) o fitxer JSON
- **Altres**: cap biblioteca externa (JS pur), PDO per a MySQL.
